# 🧭 Section 3 — Navigation de l'Application

> **Guide Poppo Live 2026** | Section 3 sur 16

## 3.1 Les 5 Onglets Principaux

| Icône | Nom | Fonction |
|-------|-----|----------|
| 🏠 | **Home** | Streams tendance, filtrage par région (drapeaux), catégories (Hot, New, Nearby) |
| 🎥 | **Live** | Démarrer un stream : Solo, Party Room, Audio Live, PK Battle |
| 🔍 | **Discover** | Partager/explorer clips, photos, Moments (type Instagram) |
| 💬 | **Messages** | Chat privé, messages système, codes vérification, alertes agence |
| 👤 | **Profil** | Wallet, Reward Center, **Creator Center (Nouveauté 2026)**, VIP, Paramètres |

> 🌟 **NOUVEAUTÉ MAI 2026 : Le Creator Center**
> Intégré dans l'onglet Profil, c'est le nouveau tableau de bord des créateurs. Il permet de :
> - Publier de courtes vidéos (type TikTok/Reels) pour attirer du trafic.
> - Suivre son **"Creator Level"** et débloquer de nouvelles statistiques analytiques.
> - Gérer les membres de son "Fan Club".

---

## 3.2 Modes de Live Disponibles

| Mode | Description | Idéal pour |
|------|-------------|-----------|
| **Solo Live** | Vous seul en direct | Chant, discussion, talents |
| **Party Room** | Salle multi-guests (jusqu'à 9) | Karaoké groupe, sessions sociales |
| **Audio Live** | Stream audio uniquement | Discussions, podcasts |
| **PK Battle** | Compétition contre un autre host | Maximiser les revenus |

### Avant de lancer un Live :
1. Vérifier connexion internet
2. Choisir un titre accrocheur
3. Ajouter des tags/hashtags pertinents
4. Vérifier éclairage et son
5. Cliquer "Go Live" 🔴

---

## 3.3 Éléments Clés dans l'Interface

| Élément | Fonction |
|---------|----------|
| **Treasure Box** | Coffre apparaissant en stream → coins gratuits (timer 5 min) |
| **Effets d'entrée** | Animations VIP visibles par tous en entrant dans un stream |
| **Bouton ❤️** | Envoyer des cœurs gratuits → booste le ranking du stream |
| **Bouton Report** | ⋮ → "Report" pour signaler du contenu inapproprié |

## 3.4 Gestes Utiles

| Geste | Résultat |
|-------|----------|
| Swipe gauche sur Home | Changer de catégorie |
| Swipe haut en live | Stream suivant |
| Swipe bas en live | Stream précédent |
| Double-tap vidéo | Envoyer un cœur |
| Tap sur pseudo viewer | Voir son profil |
